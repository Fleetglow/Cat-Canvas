# Changelog

## [Unreleased] - V1.1

### Added
- 设置面板：点击右上角齿轮图标打开设置
- F键聚焦：按 F 键可聚焦选中节点
- Ctrl/Shift 功能对换：设置中可交换 Ctrl（多选）与 Shift（切割模式）的按键功能
- X 键删除节点：Blender 风格，默认关闭，开启后按 X 删除选中节点（文本输入时自动忽略）
- 图片生成时间显示开关：设置中可选择是否在输出节点上显示图片生成耗时
- 生成日志弹窗重构：改为网格布局，支持批量操作
- Ctrl/Alt+滚轮缩放：文本/output 节点使用滚轮时为滚动，增加 Ctrl/Alt+滚轮缩放
- 空格平移：按住空格时可平移画布，与鼠标中键功能一致
- Ctrl+Y 反撤销：撤销后可恢复，与常规软件行为一致
- API 平台列表排序：支持拖拽排序

### Changed
- 帮助弹窗：画布下方的帮助说明移动到画布右上方
- 小地图缩放控件：右下角添加控件可关闭小地图
- 画布边界调整：优化画布可视区域边界
- 复制功能优化：现在可以复制多个节点，并且可以保持连线
- 图片预览优化：删除缩放最小为1的限制，UI重新设计

### Fixed
- 全局禁用浏览器缩放快捷键：Ctrl+/-/0 不再触发浏览器自带缩放
- 代理检测逻辑修复：移除自动代理检测，改为默认直连，仅用户明确配置 `INFINITE_CANVAS_OUTBOUND_PROXY` 环境变量时才启用代理（避免部分平台如 `image.pytn.blog` 走代理导致超时）
- canvas.js 重复引用修复：移除 HTML 中重复的 script 标签，修复变量重复声明错误

### Other
- Bundled Python 升级到 3.14：`python/` 目录更新为 Python 3.14 embed 版本，更新 `run.bat` 优先使用本地 `python/` 目录
- 更新静态资源版本号（2026.05.27）
