# 更新日志

## V1.1

### ✨ 新增功能

* **设置面板**：点击右上角齿轮图标打开设置
* **F键聚焦**：按 F 键可聚焦选中节点
* **Ctrl/Shift 功能对换**：设置中可交换 Ctrl（多选）与 Shift（切割模式）的按键功能
* **X 键删除节点**：Blender 风格，默认关闭，开启后按 X 删除选中节点（文本输入时自动忽略）
* **图片生成时间显示开关**：设置中可选择是否在输出节点上显示图片生成耗时
* **生成日志弹窗重构**：改为网格布局，支持批量操作
* **Ctrl/Alt+滚轮缩放**：文本/output节点使用滚轮时为滚动，增加Ctrl/Alt+滚轮缩放
* **空格平移**：按住空格时可平移画布，与鼠标中键功能一致
* **Ctrl+Y 反撤销**：撤销后可恢复，与常规软件行为一致
* **API 平台列表排序**：支持拖拽排序

### 🎨 优化

* **帮助弹窗**：画布下方的帮助说明移动到画布右上方
* **小地图缩放控件**：右下角添加控件可关闭小地图
* **画布边界调整**：优化画布可视区域边界
* **复制功能优化**：现在可以复制多个节点，并且可以保持连线
* **图片预览优化**：删除缩放最小为1的限制，UI重新设计
* **默认打开无限画布**：打开应用时默认显示"无限画布"页面
* **记住上次画布**：每次打开画布时记录，切换回无限画布页面时自动恢复上次打开的画布
* **侧边栏状态记忆**：收起/展开状态保存，下次访问时恢复
* **侧边栏重构**：侧边栏UI交互调整
* 

### 🐛 修复

* **全局禁用浏览器缩放快捷键**：Ctrl+/-/0 不再触发浏览器自带缩放

### 🗑️ 大规模功能精简

* **删除智能画布功能**：移除 smart-canvas 全部文件
* **删除 ComfyUI 功能**：移除 ComfyUI 设置页、全部工作流 JSON 文件、画布中 ComfyUI 节点系统和后端路由
* **删除侧边栏本地功能**：移除文生图/细节增强/图片编辑/角度控制四个独立页面及侧边栏入口
* **删除 ModelScope 和 RunningHub**：移除后端 API、画布 msgen 节点子系统、API 设置页中的 MS/RH 配置 UI、GPT 对话中的 MS provider 分支
* **删除英文支持**：仅保留中文界面，简化了 i18n 系统，移除所有英文翻译和语言切换功能
