# Lightbox 预览窗口重新设计

## 概述

重新设计图片预览窗口（lightbox），从当前的"左图+右侧简单提示词面板"改为"左图+右侧完整信息面板"的左右分栏布局。

## 当前状态

- 左侧：图片预览（支持缩放/平移）
- 右侧：只有提示词文本 + 复制/再次运行按钮
- 顶部：分辨率信息 + 下载/关闭按钮

## 新设计

### 布局

```
┌─────────────────────────────────────────────────────────────┐
│ [512×1536 ≈9:16 1m 02s]                    [下载] [关闭]   │
├──────────────────────────┬──────────────────────────────────┤
│                          │  提示词 📋                       │
│                          │  风格摄影，展现一种神秘的...      │
│      图片预览            │                          [展开]  │
│     (支持缩放/平移)      │──────────────────────────────────│
│                          │  参考图                          │
│                          │  [缩略图1] [缩略图2] [缩略图3]   │
│                          │──────────────────────────────────│
│                          │  参数配置                        │
│                          │  平台: RunningHub                │
│                          │  模型: gpt-image-2              │
│                          │  尺寸: 512×1536                  │
│                          │  质量: auto                      │
│                          │  格式: png                       │
│                          │  创建于 2026/5/25 10:27:20       │
│                          │  耗时 1m 02s                     │
├──────────────────────────┴──────────────────────────────────┤
│        [再次运行]          [发送到画布]          [删除记录]   │
└─────────────────────────────────────────────────────────────┘
```

### 数据来源

| 信息 | 数据来源 |
|------|----------|
| 提示词 | `meta.run.prompt` |
| 参考图 | `meta.run.refs` (从 generator 节点输入端口获取) |
| 平台 | `log.platform` 或 `runPlatformLabel(run)` |
| 模型 | `log.model` 或 `runTaskLabel(run)` |
| 尺寸 | 从图片 `naturalWidth × naturalHeight` 获取 |
| 质量/格式 | 从 `log.request` 或 `meta.run.node` 获取 |
| 耗时 | `meta.runMs` |
| 创建时间 | `log.createdAt` |

### 修改文件

1. **`static/canvas.html`** - 重构 lightbox HTML 结构
2. **`static/css/canvas.css`** - 新增/修改 lightbox CSS 样式
3. **`static/js/canvas.js`** - 更新 `openOutputLightbox`、`setupOutputPromptPanel` 等函数

### 功能点

1. **提示词区域**：长文本默认截断，点击"展开"显示完整内容
2. **参考图区域**：显示关联 generator 节点的输入参考图缩略图
3. **参数配置区域**：显示平台、模型、尺寸、质量、格式等信息
4. **底部操作按钮**：
   - 再次运行：复用现有 `rerunFromOutputMeta` 逻辑
   - 发送到画布：将图片作为节点添加到画布
   - 删除记录：删除当前输出记录
5. **顶部信息栏**：显示尺寸、比例、耗时

### 保留功能

- 图片缩放/平移（滚轮缩放 + 拖动平移）
- 对比模式（双击切换）
- 视频播放支持
- 下载功能
